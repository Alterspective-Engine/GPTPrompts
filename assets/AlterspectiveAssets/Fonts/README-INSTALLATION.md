# Chronicle Display Font Installation

## Quick Start

**Simply double-click `Install Fonts.bat`** and follow the prompts.

## What Gets Installed

This installer will add the following Chronicle Display fonts to your system:
- Chronicle Display Black
- Chronicle Display Light Italic
- Chronicle Display Roman
- Chronicle Display Semibold

## Installation Method

- **User-level installation** (no administrator rights required)
- Fonts are installed to: `%LOCALAPPDATA%\Microsoft\Windows\Fonts`
- Works on Windows 10 and Windows 11

## After Installation

1. **Close and restart** any applications that were open during installation
2. The fonts will now appear in your font list in Word, PowerPoint, etc.
3. If you had PowerPoint presentations with embedded Chronicle Display fonts, you can now:
   - Remove the restricted embedded fonts
   - PowerPoint will automatically use your installed fonts

## Troubleshooting

### Fonts don't appear after installation
- Make sure you **closed and reopened** the application
- Try restarting your computer

### "Already installed" message
- The fonts are already on your system
- No action needed

### Installation errors
- Make sure the `chronicle-display` folder with the .otf files is in the same location as the installer
- Try closing all applications and running the installer again

## Uninstalling

To remove the fonts:
1. Open: `%LOCALAPPDATA%\Microsoft\Windows\Fonts`
2. Delete the Chronicle Display .otf files
3. Open Registry Editor (regedit)
4. Navigate to: `HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts`
5. Delete the Chronicle Display entries
